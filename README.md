# 吉林省高考志愿规划系统 · 本地版 v2.0

## 快速启动

**Windows**：双击 `启动.bat`
**Mac/Linux**：`chmod +x 启动.sh && ./启动.sh`
**手动**：`pip install flask openpyxl pandas numpy && python app.py`

浏览器打开 → http://localhost:5000

---

## 使用流程

1. **填写考生信息**：分数、选科、目标专业方向、排除条件、城市偏好
2. **点击"生成方案"**：系统自动从2026年录取数据中筛选，生成冲稳保30志愿
3. **查看结果**：每个志愿显示院校/城市/层次/意向专业/安全评级/MC命中率
4. **蒙特卡洛仿真**：进入仿真页面，调节波动参数，查看每专业命中概率
5. **导出Excel**：下载3张Sheet的志愿规划表，直接用于填报

## 系统要求
- Python 3.9+（推荐3.11）
- 内存 512MB+
- 无需联网（数据完全本地）

## 规划逻辑说明
- **冲志愿**：gmin>考生分，历史冲击型，有意向专业分差≤5分，2026年降线有望命中
- **稳志愿**：sc6≤考生分，安全稳录，第6专业作调剂锚防退档
- **保志愿**：考生分超sc6≥10分，绝对兜底

## 核心安全规则
> **退档触发条件**：组最低分≤考生分（提档）+ 所有填报专业分>考生分 + 不服从调剂 = 退档 → 后续所有志愿作废！
> 
> 本系统所有志愿默认服从调剂，第6专业严格要求分数≤考生分。
